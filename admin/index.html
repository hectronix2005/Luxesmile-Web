Reseñas de Google

Las reseñas se traen automáticamente de Google Business mediante un widget de Elfsight. El botón Deja tu reseña en Google aparecerá en el sitio, en el footer y en el menú flotante.

Cómo obtenerla: entra a business.google.com → tu negocio → Pedir reseñas → copia el enlace corto que aparece (formato https://g.page/r/.../review). Si no aparece esa opción, usa el formato con tu Place ID.

Cómo obtenerlo: 1) Crea cuenta gratuita en elfsight.com. 2) Apps → Google Reviews → conecta tu Google Business Profile. 3) Personaliza la apariencia (recomendado: limit 5 reseñas, ordenar por más recientes, mostrar rating). 4) Al guardar, te darán un código <div class="elfsight-app-XXXXXXXX"></div> — solo copia la parte XXXXXXXX (sin "elfsight-app-") y pégala aquí.

Publicación online (GitHub)

El sitio lee assets/data/content.json desde el repo. Al pulsar "Publicar online" el admin hace un commit directo vía GitHub API. GitHub Pages rebuild el sitio en ~1 minuto.

Crea el token en github.com/settings/personal-access-tokens/new → Repository access: Only select repositories → elige este repo → Permissions → Contents: Read and write. Expiración recomendada: 90 días.
El token se guarda solo en este navegador; nunca se commitea al repo.

Backup local

Exporta el JSON actual (lo que ves en el formulario) a un archivo, o impórtalo para previsualizar antes de publicar. Útil como copia de seguridad manual o para mover contenido entre navegadores sin usar la API de GitHub.

Importar solo carga el JSON al formulario; recuerda pulsar "Publicar online" para que el sitio refleje los cambios.
